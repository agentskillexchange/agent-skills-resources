# Checklists

Use these short checklists when you need a practical review artifact without a
long template.

| Checklist | Use it for |
|---|---|
| [First Skill Review](first-skill-review.md) | Decide if a skill is worth deeper evaluation |
| [Source-Backed Resource Review](source-backed-resource-review.md) | Check if a resource belongs in this repo |
| [MCP Tooling Readiness](mcp-tooling-readiness.md) | Review MCP servers, tools, permissions, and setup |
| [Team Pilot Readiness](team-pilot-readiness.md) | Decide if a team pilot is ready to start |
| [Security And Permissions Review](security-and-permissions-review.md) | Review risky access before adoption |

For longer evidence capture, use the fillable [templates](../templates/).
