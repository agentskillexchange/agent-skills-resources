# Best-Practices Cookbook

This cookbook collects practical patterns for writing and reviewing agent
skills. It is framework-neutral and catalog-light: the examples teach patterns
rather than listing specific catalog entries.

| Topic | Use it for |
|---|---|
| [Source-Backed Skills](source-backed-skills.md) | Avoid vague or unsupported claims |
| [Install And Setup Clarity](install-and-setup-clarity.md) | Make first use repeatable |
| [Security And Permissions](security-and-permissions.md) | Name risks before granting access |
| [Multi-Framework Skills](multi-framework-skills.md) | Support multiple surfaces honestly |
| [Maintenance And Freshness](maintenance-and-freshness.md) | Keep resources current |
| [Common Anti-Patterns](common-anti-patterns.md) | Spot weak skills quickly |

Start with the weakest part of the skill. If setup is unclear, fix setup before
polishing wording. If permissions are unclear, fix safety before adding examples.
