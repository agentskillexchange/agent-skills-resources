This section maps popular agent ecosystems to the agent skills mental model.
It is not a ranking. Use it to understand where frameworks, SDKs, tools,
workflows, memory, evals, and MCP fit.

| CrewAI | [CrewAI](crewai.md) | Crews, flows, role-based agents, and business process automation |
| Microsoft AutoGen | [AutoGen](autogen.md) | Multi-agent research and application patterns |

| LlamaIndex | [LlamaIndex](llamaindex.md) | RAG agents, data workflows, and retrieval-backed tools |
| Pydantic AI | [Pydantic AI](pydantic-ai.md) | Typed Python agents, tool calls, evals, and graph workflows |
