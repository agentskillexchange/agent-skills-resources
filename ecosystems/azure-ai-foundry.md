# Azure AI Foundry

Azure AI Foundry is Microsoft Azure's platform for building, evaluating, and
operating AI applications and agents.

## Where It Fits

- Azure-hosted AI apps and agents
- evaluation and deployment workflows
- enterprise governance and project structure
- Microsoft ecosystem integration

## Skill-Adjacent Concepts

- agents and projects
- model and tool configuration
- evaluation
- deployment
- enterprise governance

## Best Fit

- teams already using Azure
- workflows that need Microsoft identity and governance controls
- enterprise pilots that need evaluation and deployment structure

## When Not To Use It

- local-only skill experiments
- workflows with no Azure ownership or tenant access
- lightweight tasks that do not need a managed AI platform

## Safety And Rollout

- Review tenant and project boundaries.
- Keep data handling aligned with organizational policy.
- Capture evaluation results before rollout.
- Separate sandbox and production deployments.
