# Braintrust

Braintrust provides tooling for evals, prompt management, datasets, experiments,
and observability for AI applications.

## Where It Fits

- eval tracking
- experiment comparison
- prompt and dataset management
- production feedback loops

## Skill-Adjacent Concepts

- datasets
- experiments
- traces
- prompts
- evaluation records

## Best Fit

- teams running repeated eval cycles
- comparing skill or prompt revisions
- managing evidence across pilots and production workflows

## When Not To Use It

- one-off manual review
- workflows that should remain entirely local
- cases where sensitive logs cannot be stored in the chosen platform
