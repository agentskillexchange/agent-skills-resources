# DeepEval

DeepEval is an LLM evaluation framework for writing test cases and measuring
LLM application behavior.

## Where It Fits

- LLM app test cases
- regression tests
- metrics-based evaluation
- CI-style quality checks

## Skill-Adjacent Concepts

- test cases
- metrics
- assertions
- datasets
- regression runs

## Best Fit

- teams adding evals to application workflows
- skills that produce structured or repeatable outputs
- CI checks around prompt or workflow changes

## When Not To Use It

- when the workflow cannot be expressed as repeatable test cases
- when the main risk is permissions or external side effects
- without human review for ambiguous outputs
