# GAIA

GAIA is a benchmark for general assistant tasks that often require reasoning,
tool use, and multi-step work. It is useful context for agent skill readers, but
it is broader than most individual production skills.

## Where It Fits

- general assistant evaluation
- tool-use task orientation
- multi-step reasoning examples

## Skill-Adjacent Concepts

- task decomposition
- external tool use
- source-backed answers
- answer verification

## Best Fit

- understanding broad assistant capability evaluation
- comparing general task performance with workflow-specific evals
- designing multi-step eval examples

## When Not To Use It

- as the only approval evidence for a specific skill
- for narrow deployment or permission checks
- when the workflow requires domain-specific test data
