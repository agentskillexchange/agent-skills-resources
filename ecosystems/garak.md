# garak

garak is a tool for probing and testing LLM vulnerabilities.

## Where It Fits

garak fits the red-teaming and vulnerability probing layer. Use it when a team
wants broader adversarial testing against model or application behavior.

## Skill-Adjacent Concepts

- vulnerability probing
- adversarial prompts
- model behavior testing
- risk discovery
- red-team evidence

## Best Fit

- security-focused LLM testing
- pre-rollout adversarial checks
- comparing model behavior under attack-style prompts

## When Not To Use It

- as a production monitor
- as a complete security approval
- when the workflow cannot tolerate generated adversarial test data
