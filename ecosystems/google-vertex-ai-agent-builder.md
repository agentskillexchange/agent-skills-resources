# Google Vertex AI Agent Builder

Google Vertex AI Agent Builder is a Google Cloud surface for building agent and
enterprise search experiences with managed infrastructure.

## Where It Fits

- Google Cloud agent applications
- enterprise search and conversational experiences
- managed deployment for agent-like workflows
- integration with Google Cloud project and IAM controls

## Skill-Adjacent Concepts

- managed agent building
- data grounding and enterprise search
- project/IAM configuration
- deployment and monitoring by cloud setup
- integration with Google Cloud services

## Best Fit

- teams already using Google Cloud
- enterprise search or support workflows
- managed agent pilots that need cloud governance

## When Not To Use It

- local coding-agent workflows
- workflows that do not need Google Cloud project setup
- unbounded tool automation without a clear approval model

## Safety And Rollout

- Review project and IAM scope.
- Separate test and production data sources.
- Capture deployment and evaluation evidence.
- Confirm how logs and user data are retained.
