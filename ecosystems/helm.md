# HELM

HELM is a holistic evaluation framework and benchmark effort from Stanford CRFM.
It is useful for understanding model-level evaluation dimensions and benchmark
reporting.

## Where It Fits

- model evaluation
- benchmark reporting
- scenario and metric design
- broad comparison context

## Skill-Adjacent Concepts

- scenarios
- metrics
- model behavior
- benchmark reports
- transparency around evaluation setup

## Best Fit

- readers learning benchmark design
- model-level evaluation orientation
- comparison with workflow-specific eval evidence

## When Not To Use It

- as a direct skill approval test
- for tool permission or deployment checks
- without task-specific examples and traces
