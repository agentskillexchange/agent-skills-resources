# Inspect AI

Inspect AI is an evaluation framework for large language models. It supports
programmable eval tasks, datasets, solvers, and scorers.

## Where It Fits

- custom eval tasks
- model and workflow evaluation
- reproducible scoring
- safety and behavior tests

## Skill-Adjacent Concepts

- datasets
- task definitions
- scorers
- solvers
- logs and eval records

## Best Fit

- teams that need programmable evals
- safety and behavior regression checks
- workflow-specific evaluation harnesses

## When Not To Use It

- when a manual checklist is enough
- when the workflow lacks repeatable inputs
- as a substitute for approval gates on risky actions
