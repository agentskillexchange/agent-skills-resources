# Lakera

Lakera provides security tooling for protecting generative AI applications.

## Where It Fits

Lakera fits the AI application security and runtime protection layer. It is
relevant when a team needs controls around prompt injection, unsafe content, and
LLM application risk.

## Skill-Adjacent Concepts

- prompt injection defense
- runtime protection
- policy checks
- model input/output risk
- security review

## Best Fit

- teams operating AI apps exposed to untrusted input
- workflows with sensitive tools or data
- security review around prompt injection and data leakage

## When Not To Use It

- as the only evidence for workflow safety
- for static documentation review
- when the team has not defined what should be blocked
