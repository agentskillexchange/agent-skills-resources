# OpenAI Evals

OpenAI Evals provides resources for evaluating model and task behavior through
datasets, grading, and repeatable evaluation workflows.

## Where It Fits

- OpenAI-backed model and task evaluation
- regression checks for prompts and tools
- structured eval datasets

## Skill-Adjacent Concepts

- eval datasets
- graders
- task rubrics
- model comparison
- regression records

## Best Fit

- OpenAI API workflows
- prompt and skill regression checks
- teams that need repeatable grading around OpenAI-backed agents

## When Not To Use It

- as the only evidence for external side effects
- for non-OpenAI stacks without adaptation
- when a simpler manual review is sufficient
