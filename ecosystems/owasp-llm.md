# OWASP LLM Top 10

OWASP LLM Top 10 is a public security reference for common risks in LLM
applications.

## Where It Fits

It fits the risk vocabulary and review layer. Use it to structure security
review for skills that call tools, retrieve untrusted content, or affect users.

## Skill-Adjacent Concepts

- prompt injection
- sensitive information disclosure
- supply-chain risk
- excessive agency
- insecure output handling
- model behavior review

## Best Fit

- security review checklists
- risk taxonomy for agent workflows
- training reviewers on common LLM app failures

## When Not To Use It

- as a complete local policy
- as proof a workflow is safe
- as a substitute for testing and approval gates
