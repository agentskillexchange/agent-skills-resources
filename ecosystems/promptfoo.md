# promptfoo

promptfoo is an evaluation and red-teaming tool for prompts, models, and LLM
applications.

## Where It Fits

promptfoo fits the eval and adversarial testing layer. It helps teams compare
outputs, run regression checks, and test for unsafe behavior.

## Skill-Adjacent Concepts

- eval suites
- regression tests
- red-team prompts
- prompt injection tests
- model/provider comparison
- CI checks for LLM behavior

## Best Fit

- prompt and workflow regression tests
- adversarial examples before rollout
- comparing model/provider behavior for a workflow

## When Not To Use It

- as the only approval gate for sensitive actions
- when expected behavior is not defined
- for runtime monitoring without test cases
