# PyRIT

PyRIT is Microsoft's Python Risk Identification Tooling for generative AI red
teaming and risk evaluation.

## Where It Fits

PyRIT fits the red-team automation and risk identification layer. It is useful
when teams need repeatable probes and evidence for generative AI systems.

## Skill-Adjacent Concepts

- red-team automation
- adversarial testing
- risk identification
- prompt attack workflows
- evaluation artifacts

## Best Fit

- structured red-team exercises
- security evaluation before rollout
- repeatable risk probes for agent workflows

## When Not To Use It

- as a replacement for human security review
- for workflows without defined risk categories
- where test artifacts cannot be stored safely
