# Ragas

Ragas is an evaluation framework focused on retrieval-augmented generation and
LLM application quality.

## Where It Fits

- RAG workflow evaluation
- retrieval quality checks
- grounded answer review
- dataset-backed app tests

## Skill-Adjacent Concepts

- context relevance
- answer faithfulness
- retrieval scoring
- test datasets
- experiment tracking by app setup

## Best Fit

- skills that search, retrieve, summarize, or answer from documents
- data and research workflows
- regression checks for retrieval changes

## When Not To Use It

- pure coding-agent workflows
- workflows with no retrieval or grounding component
- approval checks for external side effects
