| Path | Start here | Outcome |

| Comparing concepts | [Skills vs Tools vs MCP](skills-vs-tools-vs-mcp.md) | Know where each layer belongs |
| Designing a skill | [Skill Design Patterns](skill-design-patterns.md) | Pick a reusable structure for a workflow |
