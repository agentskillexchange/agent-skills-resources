# Beginner Track

## Who It Is For

Developers, operators, and agent users who want to understand agent skills before
choosing a framework or catalog entry.

## Expected Time

45-60 minutes.

## Prerequisites

- Basic familiarity with command-line tools or developer workflows.
- No prior MCP, agent framework, or skill authoring experience required.

## Practice Exercise

Pick one workflow you repeat weekly. Write down:

- the trigger
- the tool or API involved
- the expected output
- the safety check you would want before trusting an agent

## What Good Looks Like

- You can explain the difference between a skill, a tool, an MCP server, and an
  agent workflow.
- You can identify when a skill is useful and when plain documentation is enough.
- You know where to find official framework resources.

## Common Mistakes

- Treating popularity as proof of safety.
- Calling a generic prompt a skill.
- Ignoring setup, permissions, and verification.
