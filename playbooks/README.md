# Adoption Playbooks

These playbooks help teams evaluate skill-based agent workflows without treating
the companion repo as the canonical catalog. Use
[agentskillexchange/skills](https://github.com/agentskillexchange/skills) for
the full skill source of truth.

## Rollout Pattern

1. Sandbox: run with sample repos, fixtures, or read-only systems.
2. Limited team: use real work with narrow permissions and human approval.
3. Production: require logging, verification evidence, ownership, and rollback.

Use the [Adoption Matrix](../adoption-matrix.md) to compare starting points.
