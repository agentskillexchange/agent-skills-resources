# Starter Kits

Short paths for common readers. Each starter kit points to existing resources
and representative ASE examples without duplicating the full catalog.

## Choose By Role

| If you are... | Start with | Then use |
|---|---|---|
| Building a reusable workflow | [Skill Author](skill-author.md) | [First Skill Review Checklist](../checklists/first-skill-review.md) |
| Reviewing a submitted or internal skill | [Skill Author](skill-author.md) | [Skill Evaluation Worksheet](../templates/skill-evaluation-worksheet.md) |
| Piloting skills with a team | [Team Evaluation](team-evaluation.md) | [Team Pilot Readiness](../checklists/team-pilot-readiness.md) |
| Adopting a coding-agent workflow | [Coding Agent User](coding-agent-user.md) | [Framework Comparison](../framework-comparison.md) |
| Connecting tools through MCP | [MCP Tooling User](mcp-tooling-user.md) | [MCP Tooling Readiness](../checklists/mcp-tooling-readiness.md) |

| Starter kit | Use it when |
|---|---|
| [Coding Agent User](coding-agent-user.md) | You use Codex, Claude Code, Copilot, Cursor, Gemini CLI, or similar coding agents. |
| [MCP Tooling User](mcp-tooling-user.md) | You want skills that wrap MCP servers, tools, APIs, or data sources. |
| [Team Evaluation](team-evaluation.md) | Your team needs to evaluate skills before rollout. |
| [Skill Author](skill-author.md) | You want to write or improve a source-backed skill. |

The full catalog remains [agentskillexchange/skills](https://github.com/agentskillexchange/skills)
and [Agent Skill Exchange](https://agentskillexchange.com/browse-skills/).
